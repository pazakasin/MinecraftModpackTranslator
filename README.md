# MinecraftModpackTranslator

マインクラフトModpackの日本語翻訳ツールです。  

Modpackに含まれるクエストファイルや各Modファイルを日本語化するリソースパックを作成します。  

ツールの性質上、100万文字以上の大量の文章を翻訳APIを使用して翻訳する事になります。  
使用方法によっては大量の料金が発生する可能性があるため、くれぐれも注意の上、自己責任で使用してください。  
実行形式ファイルは配布しません、実行する場合は各自でビルドをお願いします。

使用可能な翻訳プロバイダは以下の通り。

 - Google Translation API
 - DeepL API のAPI
 - ChatGPT API（※）
 - Claude API（※）  
 
（※：実装途中。正常に機能しない可能性大）
